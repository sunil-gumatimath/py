# Python Lists Script

This script, `lists.py`, demonstrates various list operations in Python.

## Script Details (`lists.py`)

The script covers the following list concepts and operations:

- **Creating Lists**: Declaring lists with different data types, including empty lists.
- **Printing Lists**: Displaying lists and their lengths using `len()`.
- **Accessing List Items**: Using positive and negative indexing to access list elements.
- **Finding Last Index**: Calculating the last index of a list.

## How to Run

To run the script, you can use the following command in your terminal:

```bash
python lists.py
