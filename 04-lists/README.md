# Python Lists Script

This script, `lists.py`, demonstrates various list operations in Python.

## Script Details (`lists.py`)

The script covers the following list concepts and operations:

- **Creating Lists**: Declaring lists with different data types, including empty lists with type hints.
- **Printing Lists**: Displaying lists and their lengths using `len()`.
- **Accessing List Items**: Using positive and negative indexing to access list elements.
- **Finding Last Index**: Calculating the last index of a list.
- **Slicing Lists**: Extracting sublists using slice notation with positive and negative indices.
- **Modifying List Items**: Updating list elements by index using both positive and negative indices.
- **Checking Membership**: Using the `in` operator to check if an item exists in a list.
- **Appending Items**: Adding items to the end of a list using the `append()` method.
- **Inserting Items**: Inserting items at specific positions using the `insert()` method.

## How to Run

To run the script, you can use the following command in your terminal:

```bash
python lists.py
```

## Examples Covered

The script demonstrates:
- Creating lists of fruits, vegetables, animal products, web technologies, and countries
- Accessing first, second, and last elements
- Using negative indexing to access elements from the end
- Slicing with various start and end indices
- Modifying list elements by index
- Checking if items exist in lists
- Adding items to lists using `append()` and `insert()`
