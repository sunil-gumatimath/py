# Python Operations Script

This script, `operations.py`, provides a comprehensive demonstration of various operations in Python, including arithmetic, comparison, and logical operations.

## Script Details (`operations.py`)

The script covers the following topics:

*   **Arithmetic Operations**: Addition, subtraction, multiplication, division, modulus, floor division, and exponentiation.
*   **Data Types**: Operations on integers, floating-point numbers, and complex numbers.
*   **Variable Assignments**: Declaring variables and assigning them the results of arithmetic operations.
*   **Comparison Operations**: `==`, `!=`, `>`, `<`, `>=`, and `<=`.
*   **Logical Operations**: `and`, `or`, and `not`.
*   **Identity and Membership Operators**: `is`, `is not`, `in`, and `not in`.

## How to Run

To run the script, you can use the following command in your terminal:

```bash
python operations.py
```
